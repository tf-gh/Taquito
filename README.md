# Taquito 
## Introduction 
Taquito is a TypeScript library suite for development on the Tezos blockchain.

See [Taquito](https://tezostaquito.io) for more information.

## Security assessment
Tezos Foundation has requested [Least Authority](https://leastauthority.com) to perform a security assessment of Taquito.

The final report of Least Authority's security assessment is: 
- [Security Audit Report – Taquito](Least%20Authority%20-%20Tezos%20Foundation%20Taquito%20Final%20Audit%20Report.pdf)
